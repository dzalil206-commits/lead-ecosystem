📋 Как начать работу
1
Купите лицензию на странице тарифов — доступны Miner, Sender и комбо-пакеты
2
Скачайте приложение — кнопка выше станет активной после загрузки файла на сервер
3
Введите лицензионный ключ — найдёте его в личном кабинете → раздел «Мои лицензии»
4
Для Miner: введите API ID, API Hash и номер телефона — получить на my.telegram.org/apps
5
Укажите чат или канал для анализа — приложение выгрузит аудиторию в файл
⚖️

Важно. Приложения предназначены для легальной коммуникации с аудиторией, давшей согласие на получение сообщений. Использование должно соответствовать законодательству РФ, правилам Telegram и ФЗ «О рекламе». Ответственность за характер использования лежит на пользователе.
