1. Оператор персональных данных

Настоящая Политика конфиденциальности разработана в соответствии с требованиями Федерального закона от 27.07.2006 № 152-ФЗ «О персональных данных» и описывает порядок обработки персональных данных Пользователей сервиса TG Lead Wareon (далее — «Сервис»).

Оператор персональных данных: администрация Сервиса TG Lead Wareon.

Контакт для запросов по персональным данным: {{ SUPPORT_USERNAME }}

2. Какие данные мы собираем и зачем

Мы собираем только данные, необходимые для работы Сервиса:

Категория данных Что именно Цель Срок хранения
Учётные данные Email-адрес, имя (необязательно) Идентификация, вход в аккаунт До удаления аккаунта + 30 дней
Учётные данные Telegram Номер телефона, API ID, API Hash Подключение Telegram-аккаунта Пользователя До удаления аккаунта

Технические данные IP-адрес, User-Agent, дата/время входа Безопасность, защита от мошенничества 90 дней
Данные использования Статистика запусков Miner, количество задач Отображение аналитики в кабинете До удаления аккаунта

3. Данные, которые мы НЕ собираем

Сервис не собирает, не хранит и не обрабатывает на своих серверах персональные данные третьих лиц — участников Telegram-чатов и каналов, которые Пользователь анализирует с помощью инструментов Сервиса.

Мы также не собираем:

  • Данные платёжных карт (они обрабатываются напрямую платёжными системами — ЮKassa и др.);
  • Паспортные данные и ИНН;
  • Биометрические данные;
  • Содержимое переписки в Telegram-аккаунтах Пользователей;
  • Данные о местоположении.

4. Правовые основания обработки

Обработка персональных данных Пользователей осуществляется на следующих правовых основаниях (ст. 6 152-ФЗ):

  • Согласие субъекта — Пользователь, регистрируясь на Сервисе, даёт согласие на обработку своих персональных данных;
  • Исполнение договора — обработка необходима для исполнения пользовательского соглашения (предоставление доступа к Сервису, выставление счетов);
  • Законный интерес — обеспечение безопасности Сервиса и защита от мошенничества.

5. Передача данных третьим лицам

Мы не продаём и не передаём персональные данные Пользователей третьим лицам, за исключением следующих случаев:

  • Платёжные системы — передача данных, необходимых для обработки платежей (ЮKassa и др.);
  • Хостинг-провайдер — данные хранятся на серверах провайдера в рамках договора на обработку данных;
  • Требования закона — по запросу уполномоченных органов государственной власти в установленном законом порядке.

При передаче данных субподрядчикам мы заключаем с ними соглашения, обязывающие их соблюдать настоящую Политику.

6. Хранение и защита данных

Для защиты персональных данных Пользователей Сервис применяет следующие меры:

  • Хеширование паролей (bcrypt/werkzeug) — пароли никогда не хранятся в открытом виде;
  • Шифрование соединения (HTTPS/TLS) для всех страниц Сервиса;
  • Ограниченный доступ к базе данных — только для администратора Сервиса;
  • Хранение секретных ключей в переменных окружения, а не в коде;
  • Регулярное резервное копирование данных.

Данные Пользователей хранятся на серверах, расположенных в Российской Федерации, в соответствии с требованиями ст. 18.1 152-ФЗ.

7. Файлы cookie

Сервис использует технические файлы cookie, необходимые для работы:

  • session — хранит данные авторизованной сессии (удаляется при закрытии браузера или выходе из аккаунта);
  • is_admin — флаг сессии администратора.

Мы не используем аналитические или рекламные cookie. Вы можете запретить сохранение cookie в настройках браузера, однако это сделает вход в аккаунт невозможным.

8. Права пользователей (152-ФЗ)

В соответствии с Федеральным законом № 152-ФЗ Пользователь вправе:

  • Получить информацию об обработке своих персональных данных;
  • Исправить неточные или устаревшие данные (через настройки профиля или обращение в поддержку);
  • Удалить свои данные — по запросу в поддержку, данные удаляются в течение 30 дней;
  • Отозвать согласие на обработку данных (влечёт прекращение доступа к Сервису);
  • Подать жалобу в Роскомнадзор, если Пользователь считает, что его права нарушены.

Для реализации прав обратитесь в поддержку: {{ SUPPORT_USERNAME }}. Мы рассмотрим обращение в течение 30 дней.

9. Изменение политики

Администрация вправе вносить изменения в настоящую Политику конфиденциальности. Актуальная версия всегда доступна по адресу tgleadwareon.ru/privacy. О существенных изменениях Пользователи уведомляются по email или через интерфейс Сервиса.

Продолжение использования Сервиса после вступления изменений в силу означает согласие с новой редакцией Политики.
